A 25-second clip of a claymation character walking forward through a forest. We steer the model to ask for humans walking across at the t=15s point.


We use training-free [diffusion-forcing](https://www.boyuan.space/diffusion-forcing/) style sliding latent-token continuation, aggressive [Magcaching](https://arxiv.org/abs/2506.09045) and extensive kernel fusion to achieve realtime streaming of 576p15 audio-video on an H100, and the video can be steered with text prompts. 
The aim is to:
- First, build a serving stack for fast Diffusion Transformers rollout. This is where we are now.
- Second, via [VLA-Adapter](https://vla-adapter.github.io)-style training on video-game data, produce a playable, realtime action-steerable model.
- Third, figure out persistent World Memory, possibly through a [method like this](https://wiqzard.github.io/composition-of-memory-experts/#experiments).

An realtime 576p15 LTX2.3-distilled runtime and serverstack, optimized for serving on an H100, aggressive caching, fused kernels, and diffusion-forcing style latent token continuation for unbounded duration videos.
But importantly: prompt-steering
